SOBRE NUESTRO TRABAJO

Nosotros les hablaremos acerca de nuestro aprendizaje con HTML, CSS Y JAVA mediante nuestro proyectos y conocimiento que hemos adquerido durante nuestro primer año de bachillerato.


¿Que hemos hecho?

En nuestro proyecto realizamos una página web en la que detallamos el resumen del trabajo realizado durante el año, mostrando los distintos proyectos que desarrollamos y explicando el proceso de cada uno. Además, incluimos una sección donde presentamos los lenguajes de programación que sabemos utilizar, como HTML, CSS y JavaScript, junto con los programas y herramientas que empleamos para crear nuestras páginas y materiales digitales.

También elaboramos otra página dedicada a enseñar cómo crear una página web, explicando de forma clara y práctica el uso de estos lenguajes.


¿Como lo hicimos?

En nuestro proyecto realizamos una página web utilizando HTML, CSS y JavaScript, aplicando los conocimientos adquiridos a lo largo del año. Para mejorar el resultado y lograr un trabajo más completo, nos apoyamos en tutoriales de YouTube, los cuales nos ayudaron a comprender mejor ciertas funciones y a perfeccionar el diseño y la estructura de la página. Gracias a este proceso, pudimos crear una base sólida para futuros proyectos web, combinando el aprendizaje práctico con la creatividad y el trabajo en equipo para obtener un resultado funcional y visualmente atractivo. A todo esto ¿que logramos? :

  • Creación de una página web completa y funcional
  • Implementación de un diseño moderno
  • Mejora de habilidades en programación
  • Colaboración efectiva entre los integrantes
  • Aplicación práctica de los conocimientos aprendidos
  • Desarrollo de la creatividad y resolución de problemas

Resumen del Proyecto

La creación de nuestro proyecto para la feria de logros fue un proceso completo y colaborativo, que comenzó con la planificación de la estructura y los objetivos de la página web. Primero definimos la temática y el propósito del sitio, organizando las secciones que incluiría y distribuyendo las tareas entre los integrantes del grupo. Luego, diseñamos un boceto inicial que nos permitió visualizar cómo queríamos que se viera la página. Una vez listo el plan, iniciamos la codificación utilizando HTML para la estructura base, asegurándonos de mantener un diseño claro y funcional.

Posteriormente, integramos CSS para dar estilo a la página, aplicando colores, fuentes y animaciones que hicieran más atractivo el diseño visual. También incorporamos JavaScript para añadir interactividad, como menús desplegables, botones dinámicos y cambios de tema. Durante el desarrollo, realizamos pruebas constantes, corrigiendo errores y mejorando la experiencia del usuario. Finalmente, el proyecto fue resultado de un trabajo en equipo que combinó la creatividad, la organización y los conocimientos técnicos, reflejando todo el esfuerzo invertido desde la planificación hasta la presentación final en la feria.

Herramientas de Trabajo Dominadas

SUBLIME TEXT

Editor de código fuente multiplataforma, conocido por su velocidad, eficiencia y su gran cantidad de atajos de teclado y extensiones para desarrollo.

VISUAL STUDIO

Entorno de Desarrollo Integrado (IDE) de Microsoft, usado para construir aplicaciones, sitios web y servicios con diversos lenguajes de programación, incluyendo C# y C++.

PHOTOSHOP

Software de edición de imágenes y diseño gráfico líder en la industria. Permite manipular fotografías, crear composiciones digitales y diseñar elementos visuales.

MACROMEDIA (Dreamweaver/Flash)

Conjunto de herramientas clásicas para el diseño web y la animación. Se incluye Dreamweaver (diseño HTML visual) y Flash (creación de animaciones y contenido interactivo).

NOTEPAD ++

Editor de texto y de código fuente ligero para Windows. Soporta resaltado de sintaxis y múltiples lenguajes, ideal para edición rápida de archivos.

PROGRAMAS DE WINDOWS (Office)

Dominio de las herramientas de productividad esenciales: Word (creación de documentos), Excel (análisis de datos y hojas de cálculo) y PowerPoint (diseño de presentaciones).
